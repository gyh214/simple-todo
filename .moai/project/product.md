## 1. 프로젝트 비전

### 1.1 목표

Simple ToDo는 Windows 데스크톱 환경에서 작동하는 미니멀한 할일 관리 애플리케이션입니다.
사용자의 업무 중에도 화면의 1/4 크기로 운영되어, 부담 없이 할일을 추적하고 관리할 수 있습니다.

**핵심 목표:**
- 간결함: 불필요한 복잡성 제거
- 효율성: 빠른 입출력과 응답성
- 신뢰성: 자동 저장과 백업으로 데이터 보호

### 1.2 사용자 페르소나

- **주요 사용자**: 지식 근로자, 개발자, 학생
- **사용 환경**: Windows 10+ 데스크톱
- **사용 빈도**: 일상적 (매일 여러 번)
- **필요한 기능**: 빠른 입력, 우선순위 관리, 상태 추적

### 1.3 핵심 가치 제안

1. **미니멀한 UI**: 화면의 1/4만 차지하여 다른 작업 방해 최소
2. **즉시 접근성**: 시스템 트레이에서 클릭 한 번으로 실행
3. **자동 저장**: 모든 변경사항이 즉시 저장되어 데이터 손실 방지
4. **스마트 정렬**: 납기일 기반 정렬로 우선순위 자동 관리

## 2. 현재 기능 (v2.4)

### 2.1 Todo 관리

- **생성**: Enter 키 또는 추가 버튼으로 간편한 생성
- **수정**: 더블클릭으로 편집 다이얼로그 열기
- **삭제**: 호버 시 ✕ 버튼으로 즉시 삭제
- **완료 처리**: 체크박스로 진행중 ↔ 완료 섹션 이동

### 2.2 SubTask 관리

- **생성**: 편집 다이얼로그에서 하위 할일 추가
- **펼침 상태 유지**: 완료/삭제 후에도 펼친 상태 유지
- **독립적 관리**: 각 SubTask 독립적으로 체크 가능

### 2.3 시간 관리 (납기일, 반복)

- **납기일 설정**: 캘린더 UI로 날짜 선택
- **시각적 구분**:
  - 만료 (지난 날짜): 빨간색 배경
  - 오늘: 주황색 배경
  - 임박 (3일 이내): 연한 주황색 배경
- **정렬**: 납기일 빠른순/늦은순 드롭다운

### 2.4 UI/UX 기능

- **다크 모드**: Claude 주황색(#CC785C) 강조색 사용
- **드래그 앤 드롭**: ☰ 핸들로 순서 변경
- **섹션 분할바**: 진행중/완료 섹션 비율 조정 (최소 10%)
- **키보드 단축키**: Enter(추가), Esc(닫기), Ctrl+N(새 항목), F2(편집)

### 2.5 시스템 통합

- **시스템 트레이**: 최소화 시 트레이로 이동
- **항상 위**: 토글 기능으로 다른 창 위에 고정
- **단일 인스턴스**: 중복 실행 방지 (포트 65432 기반)
- **링크/경로 인식**: URL과 파일 경로 자동 감지 및 실행

## 3. 로드맵 (우선순위 기반)

### Phase 2: 테스트 및 안정화
- **SPEC-001**: 테스트 프레임워크 구축 (pytest, 목표 85%+ 커버리지)
- **SPEC-002**: 자동 업데이트 기능 (GitHub Releases 연동)
- **SPEC-003**: 에러 핸들링 개선 (사용자 피드백 강화)

### Phase 3: 클라우드 동기화
- **SPEC-007**: 클라우드 저장소 연동 (OneDrive, Google Drive)

### Phase 4: 기능 확장
- **SPEC-005**: 협업 기능 (공유, 댓글)
- **SPEC-006**: 고급 필터링 (태그, 카테고리)
- **SPEC-008**: 모바일 앱 (iOS/Android)
