### 1️⃣ **main-window.png** — 메인 화면 (최우선)
**목적**: README 상단에 배치할 대표 이미지로, 앱의 첫인상을 결정합니다.

**촬영 방법**:
1. Simple ToDo 앱 실행
2. **진행중(Pending)** 섹션에 다음과 같이 TODO 추가:
   - "프로젝트 보고서 작성" (납기일: 내일, 색상: 주황색)
   - "회의 자료 준비" (납기일: 2일 뒤, 색상: 주황색)
   - "코드 리뷰" (납기일: 일주일 뒤, 색상: 파란색)
3. **완료(Completed)** 섹션에 TODO 추가:
   - "메일 답장" (체크 표시)
4. 메인 창을 선택 (포커스)
5. **Windows 캡처 도구** (Win+Shift+S)로 전체 창 촬영
   - 또는 Alt+Print로 활성 창 캡처 후 Paint에서 편집

**저장 위치**: `docs/images/screenshots/main-window.png`

**팁**:
- 창 크기: 기본 크기 (약 420x600)
- 배경: 깔끔한 바탕화면
- 포맷: PNG 형식
- 해상도: 1080p 이상 권장

**예상 화면**:
```
┌─────────────────────────────────┐
│ Simple ToDo                   [─][□][x] │
├─────────────────────────────────┤
│ 진행중                                    │
│ ☐ 프로젝트 보고서 작성   [내일]      │
│ ☐ 회의 자료 준비         [2일뒤]      │
│ ☐ 코드 리뷰              [7일뒤]      │
│                                       │
│ 완료                                    │
│ ☑ 메일 답장                          │
└─────────────────────────────────┘
```

### 2️⃣ **edit-dialog.png** — 편집 다이얼로그
**목적**: 주요 기능(하위할일, 반복 설정 등)을 한눈에 보여줍니다.

**촬영 방법**:
1. 메인 화면에서 첫 번째 TODO ("프로젝트 보고서 작성") **더블클릭**
2. 편집 다이얼로그 열기
3. 다음과 같이 채우기:
   - **내용**: "프로젝트 보고서 작성" (이미 있음)
   - **납기일**: 내일 (이미 설정됨)
   - **하위 할일** 추가:
     - "데이터 수집"
     - "차트 작성"
   - **반복 설정**: "매주" 선택
4. 다이얼로그 전체 창을 캡처

**저장 위치**: `docs/images/screenshots/edit-dialog.png`

**팁**:
- 모든 필드가 보이도록 스크롤 조정
- 다이얼로그 크기: 기본값 유지 (약 500x600)
- 하위 할일이 2개 모두 보여야 함

**예상 화면**:
```
┌──────────────────────────┐
│ TODO 편집               [x] │
├──────────────────────────┤
│ 내용:                        │
│ [프로젝트 보고서 작성      ] │
│                              │
│ 납기일: [2025-11-10   ▼]   │
│                              │
│ 반복:   [매주        ▼]     │
│                              │
│ 하위 할일:                  │
│ ☐ 데이터 수집              │
│ ☐ 차트 작성                │
│ [+ 추가]                     │
│                              │
│      [저장]  [취소]         │
└──────────────────────────┘
```

### 5️⃣ **system-tray.png** — 시스템 트레이
**목적**: 시스템 통합 기능을 보여줍니다.

**촬영 방법**:
1. Simple ToDo 실행 상태 유지
2. 화면 우측 하단의 **시스템 트레이** 영역 확인
3. Simple ToDo 아이콘 표시되는 영역 캡처
4. (선택) 트레이 아이콘 **우클릭**하여 메뉴 표시 후 캡처 (추가 보너스)

**저장 위치**: `docs/images/screenshots/system-tray.png`

**팁**:
- 화면 우측 하단 부분만 캡처 (약 200x100 크기)
- Simple ToDo 아이콘이 명확히 보여야 함

**예상 화면**:
```
시스템 트레이 근처에:
[시계 10:30] [음량 아이콘] [SimpleTodo 아이콘] [⬆ 확장]
```

## 🛠️ 스크린샷 촬영 도구

### Windows 기본 도구
**Windows 캡처 도구** (권장):
```
1. Win + Shift + S 누르기
2. 캡처 영역 선택 또는 전체 창 선택
3. 자동으로 클립보드에 복사됨
4. Paint나 이미지 에디터에 붙여넣기
5. PNG 형식으로 저장
```

### 단축키
- **Win + Print**: 전체 화면 캡처 (Pictures 폴더에 자동 저장)
- **Alt + Print**: 활성 창만 캡처 (클립보드에 복사)
- **Win + Shift + S**: 영역 선택 캡처 (권장)

### 이미지 편집
- **Paint** (기본 내장): 간단한 편집 및 저장
- **Snagit** (유료): 전문적인 스크린샷
- **ShareX** (무료): 다양한 편집 기능

## ❓ 문제 해결

### Q: 스크린샷이 흐릿합니다
**A**:
- Windows 캡처 도구에서 "전체 화면 캡처" 선택
- 해상도 확인: 설정 → 시스템 → 디스플레이 → 해상도 (1080p 이상)

### Q: 특정 영역만 캡처하고 싶습니다
**A**:
- Win + Shift + S로 영역 선택 후 캡처
- Paint에서 필요한 부분만 자르기

### Q: 파일이 저장되지 않습니다
**A**:
- `docs/images/screenshots/` 폴더 존재 확인
- 파일 이름에 특수문자 없는지 확인
- PNG 형식으로 저장되었는지 확인

### Q: 앱의 다크 테마가 보이지 않습니다
**A**:
- 앱 메뉴 → 설정 → 테마 → "Dark" 선택
- 앱 재시작
