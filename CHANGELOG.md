## [2.4] - 2025-01-08

### Added
- **BaseTask 추상 클래스**: Todo와 SubTask의 공통 기능 통합
  - 중복 코드 114줄 제거
  - 메서드 재사용으로 유지보수성 향상
- **DatePickerDialog 재사용 컴포넌트**: 날짜 선택 다이얼로그 표준화
  - 중복 코드 66줄 제거
  - 일관된 날짜 선택 경험 제공
- **TodoService 헬퍼 메서드**: 공통 로직 통합
  - 중복 코드 16줄 제거

### Changed
- **하위 할일 펼침 상태 유지**: UI 상태 관리 개선
  - TODO 완료 또는 삭제 후에도 펼친 하위 할일 상태 유지
  - 연속 작업 흐름 향상
- **삭제 모달 제거**: 즉시 삭제 방식으로 변경
  - 사용자 확인 모달 제거
  - 더 빠른 작업 흐름 지원
- **EditDialog 스크롤 개선**: 하위 할일이 많을 때 접근성 향상
  - 모든 입력 필드에 접근 가능
  - 스크롤 레이아웃 개선

### Fixed
- PyQt6 시그널/슬롯 패턴 최적화
- 로깅 강화로 디버깅 개선
- 코드 재사용성 및 가독성 향상

### Technical
- **DRY 원칙 적용**: 총 246줄의 중복 코드 제거
- **CLEAN Architecture 원칙 유지**: 레이어 간 의존성 관리
- **코드 품질 개선**: 더 나은 유지보수성과 확장성

## [2.2] - 2024-11-20

### Added
- **하위 할일(SubTask) 기능**: 복잡한 작업을 단계별로 관리
  - 하위 할일 추가/수정/삭제
  - 펼침/접기 토글
  - 진행 상태 표시 (예: 2/5)
- **정렬 옵션**: 4가지 정렬 방식 지원
  - 생성 순서
  - 납기일 빠른순/늦은순
  - 커스텀 순서 (드래그 앤 드롭)

### Changed
- **섹션 비율 조정**: 진행중/완료 섹션 비율 동적 조정
  - 분할바 드래그로 조정
  - 설정 자동 저장

### Fixed
- 스크롤 동작 개선
- 시스템 트레이 통합 안정화

## [2.0] - 2024-09-01

### Added
- **CLEAN Architecture 도입**: 체계적인 레이어 분리
  - Domain, Application, Infrastructure, Presentation 레이어

### Changed

  - 구형 포맷에서 새 포맷으로 자동 변환

### Technical
- PyQt6 업그레이드

## 버전 관리 정책

### Semantic Versioning
- **MAJOR** (X.0.0): 호환되지 않는 변경
- **MINOR** (0.X.0): 하위 호환 기능 추가
- **PATCH** (0.0.X): 하위 호환 버그 수정

### 릴리스 일정
- **메이저 업데이트**: 분기별 (3개월)
- **마이너 업데이트**: 월별
- **패치 업데이트**: 필요 시 수시
